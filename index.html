O que é Hermes?

Hermes é um projeto inovador de assistente pessoal baseado em Arduino, projetado para trazer inteligência e praticidade para o seu cotidiano. Com recursos personalizáveis e uma interface intuitiva, Hermes se adapta às suas necessidades, tornando a automação de tarefas e a gestão do tempo mais acessíveis do que nunca.

Desde lembretes inteligentes até o controle de dispositivos, Hermes está aqui para simplificar seu dia a dia.

Tutorial de instalação "Hermes"

  1. 1.

    Clone o repositório

    git clone https://github.com/hermes-assitente/processador.git
    Baixe o código do Hermes do GitHub para o seu computador
  2. 2.

    Instale as dependências

    pip install PyAudio google-genai simpleaudio
    Instale os pacotes Python necessários para o funcionamento
  3. 3.

    Execute o programa

    python app.py
    Inicie o Hermes executando o arquivo principal
